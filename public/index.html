Max size file is 10MB.

Information

You can upload file with curl:

curl -F "file=@test.txt" http://temp.sh/upload

With Metadata:

curl -F "file=@test.txt" -H "X-Meta-Key: Value" http://temp.sh/upload

With Duration:

curl -F "file=@test.txt" http://temp.sh/upload?d=1h

Duration can be: 5m, 1h, 1d, 1w. Default is 1h.
